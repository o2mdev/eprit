iradon_GetFBPImageType

Return numerical and string representation of the image type

Syntax:

[nimtype, simtype] = iradon_GetFBPImageType(imtype)

Description:

imtype - image type (int, 1 to 14) or - (string, 'XB'/'YB'/'ZB'/'XYB'/'XZB'/'YZB'/'XYZB'/'X'/'Y'/'Z'/'XY'/'XZ'/'YZ'/'XYZ') nimage - image type (int, 1 to 14) simtype - image type (string, 'XB'/'YB'/'ZB'/'XYB'/'XZB'/'YZB'/'XYZB'/'X'/'Y'/'Z'/'XY'/'XZ'/'YZ'/'XYZ')
